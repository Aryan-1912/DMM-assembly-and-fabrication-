# Elenco M-1008K Multimeter Kit Build

A hand-assembled digital multimeter built from a component-level kit as part of first-semester Electronics coursework at Sheridan College.

---

## Project Overview

This project involved the full assembly, soldering, and validation of the **Elenco M-1008K** digital multimeter from raw components. The objective was to develop foundational skills in PCB assembly, hand soldering, and electronic test instrument operation.

---

## Skills Demonstrated

| Skill | Details |
|---|---|
| Through-Hole Soldering | Populated and soldered all components onto the PCB by hand |
| Component Identification | Resistors, capacitors, diodes, transistors, IC chips |
| PCB Assembly | Followed assembly documentation and component placement diagrams |
| Drilling | Drilled enclosure holes for ports and hardware mounting |
| Functional Validation | Verified meter accuracy against known voltage/resistance references |
| Test Report | Documented readings and confirmed instrument met specs |
| Test Equipment Operation | Used the completed meter to perform live measurements |

---

## Instrument Specifications

| Parameter | Range |
|---|---|
| DC Voltage (DCV) | 200mV, 2000mV, 20V, 200V, 600V |
| AC Voltage (ACV) | 200V, 600V |
| DC Current (DCA) | 200µA, 2000µA, 20mA, 200mA, 10A |
| Resistance (Ω) | 200Ω, 2000Ω, 20kΩ, 200kΩ, 2000kΩ |
| Transistor Test | hFE (NPN/PNP) |
| Diode Test | Yes |
| Display | 3.5-digit LCD |

---

## Tools Used

- Soldering iron & solder
- Drill press
- Digital calipers
- Reference multimeter (for validation)
- Assembly manual / schematic

---

## Outcome

Successfully assembled a fully functional digital multimeter verified against reference measurements. This project established hands-on competency in PCB population, soldering technique, enclosure assembly, and electronic instrument validation — skills directly applicable to electronics technician and testing roles.
